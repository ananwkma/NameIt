# Plan 07-01: Name All LoL Champions Game

## Goal
Add a "Name All LoL Champions" game mode where players guess all 172 champions in any order, with a count-up timer and an alphabetical board that populates in place as champions are found.

## Context

**Existing assets to reuse:**
- `src/data/allowlist-lol.json` — 172 champions, each `{ name, aliases, platform }`. Already sorted alphabetically (Aatrox → Zeri).
- `WikidataService.searchAllowlist(input, 'lol', true)` — strict allowlist match, returns `{ name, id }` or null. Already used by `AZGameScreen`.
- `formatTime(ms)` utility — MM:SS format.
- Framer Motion (`motion`) — already a dependency.
- CSS classes — `master-container`, `game-top`, `game-title`, `header-right`, `timer-display`, `timer-icon`, `input-group`, `input-wrapper`, `error-message`, `modal-overlay`, `modal`, `victory-modal`, `final-score`, `big-number`, `action-buttons` — all already defined.

**What's different from existing games:**
- `AZGameScreen`: sequential (must do A before B), only one slot per letter. This game: free order, one slot per champion (172 total).
- `GameScreen` (100 LoL Champions): no alphabetical board, has time limit. This game: full board, no time limit.

### Input / submission

On submit:
1. Trim and lowercase input
2. Call `WikidataService.searchAllowlist(name, 'lol', true)`
3. If null → `SET_ERROR "Not a recognized LoL champion."`
4. If already guessed → `SET_ERROR "${name} already found!"`
5. Otherwise → `GUESS_CORRECT` with the canonical name
