  <action>
Create src/hooks/useLeaderboard.ts. This hook manages the full leaderboard lifecycle for a victory screen.

Interface:
```typescript
interface UseLeaderboardReturn {
  entries: LeaderboardEntry[];      // top-5 rows from DB (empty if unavailable)
  loading: boolean;                  // true while fetching
  unavailable: boolean;              // true when Supabase is null (env vars absent)
  qualifies: boolean;                // true if myTimeMs beats the 5th-place time (or <5 entries)
  playerRank: number | null;         // 1-based rank of submitted entry (null before submit)
  submitName: (name: string) => Promise<void>;  // triggers insert + re-fetch
  submitted: boolean;                // true after successful name submit
}
```

Hook signature:
```typescript
export function useLeaderboard(gameId: string, myTimeMs: number): UseLeaderboardReturn
```

Implementation:

1. Import fetchLeaderboard, submitLeaderboardEntry, LeaderboardEntry from '../services/supabase'; import supabase from '../services/supabase'.

2. State: entries (LeaderboardEntry[]), loading (bool), unavailable (bool), submitted (bool), playerRank (number | null).

3. useEffect on mount (deps: [gameId, myTimeMs]):
   - Set loading: true
   - If supabase is null: set unavailable: true, loading: false, return
   - Call fetchLeaderboard(gameId)
   - Set entries to result, set loading: false
   - Derive qualifies: result.length < 5 || myTimeMs < result[result.length - 1].time_ms

4. qualifies: derived as a computed value from entries and myTimeMs (recalculate after submit too):
   - qualifies = !unavailable && !submitted && (entries.length < 5 || myTimeMs < entries[entries.length - 1].time_ms)

5. submitName(name: string):
   - Call submitLeaderboardEntry(gameId, name, myTimeMs)
   - Re-fetch: const updated = await fetchLeaderboard(gameId)
   - Compute playerRank: find position of myTimeMs in updated list (1-based), accounting for ties using indexOf on time_ms
   - Set entries to updated, submitted: true, playerRank

6. Return all state fields plus qualifies computed value.

Edge cases:
- If fetchLeaderboard returns [] (Supabase available but table empty): qualifies = true (they would be rank 1), unavailable = false
- If fetchLeaderboard throws or times out: entries stays [], unavailable = false (Supabase was configured but errored — show empty, not "unavailable")
- Only set unavailable: true when supabase singleton is null (env vars absent)

```tsx
{/* LEADERBOARD */}
{state.status === 'WIN' && (
  <div className="leaderboard-section">
    {loading && <p className="leaderboard-loading">Loading leaderboard…</p>}
    {unavailable && <p className="leaderboard-unavailable">Leaderboard unavailable</p>}
    {!loading && !unavailable && (
      <>
        {entries.length > 0 && (
          <table className="leaderboard-table">
            <thead>
              <tr><th>#</th><th>Name</th><th>Time</th></tr>
            </thead>
            <tbody>
              {entries.map((entry, i) => (
                <tr
                  key={i}
                  className={submitted && playerRank === i + 1 ? 'leaderboard-row-mine' : ''}
                >
                  <td>{i + 1}</td>
                  <td>{entry.player_name}</td>
                  <td>{formatTime(entry.time_ms, true)}</td>
                </tr>
              ))}
            </tbody>
          </table>
        )}
        {qualifies && !submitted && (
          <div className="leaderboard-entry">
            <p>You made the top 5! Enter your name:</p>
            <form onSubmit={async (e) => {
              e.preventDefault();
              if (nameInput.trim()) await submitName(nameInput.trim());
            }}>
              <input
                type="text"
                value={nameInput}
                maxLength={5}
                onChange={(e) => setNameInput(e.target.value.toUpperCase())}
                placeholder="XXXXX"
                autoFocus
              />
              <button type="submit" disabled={!nameInput.trim()}>Submit</button>
            </form>
          </div>
        )}
        {submitted && playerRank && (
          <p className="leaderboard-rank-confirm">You placed #{playerRank}!</p>
        )}
      </>
    )}
  </div>
)}
```
