Create src/services/supabase.ts with:

1. Null-safe singleton (from RESEARCH.md Pattern 1):
   - Read VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY from import.meta.env
   - Export `supabase: SupabaseClient | null` — null when either env var is absent
   - Do NOT throw on missing env vars — the null path is the normal local-dev path

2. LeaderboardEntry type:
   ```typescript
   export interface LeaderboardEntry {
     player_name: string;
     time_ms: number;
     created_at: string;
   }
   ```

3. fetchLeaderboard(gameId: string): Promise<LeaderboardEntry[]>
   - Guard: if (!supabase) return []
   - Query: .from('leaderboard').select('player_name, time_ms, created_at').eq('game_id', gameId).order('time_ms', { ascending: true }).limit(5)
   - On error: console.warn('[Supabase] fetchLeaderboard error:', error.message) and return []
   - Wrap the whole call in Promise.race with a 5-second timeout that resolves to [] on expiry (handles free-tier cold-start pauses)

4. submitLeaderboardEntry(gameId: string, playerName: string, timeMs: number): Promise<boolean>
   - Guard: if (!supabase) return false
   - Insert: { game_id: gameId, player_name: playerName.toUpperCase().slice(0, 5), time_ms: timeMs }
   - On error: console.warn and return false; on success: return true

5. saveLlmAllowlistEntry(categoryId: string, inputNormalized: string, canonicalName: string, description: string): Promise<void>
   - Guard: if (!supabase) return
   - Upsert with onConflict: 'category_id,input_normalized', ignoreDuplicates: true
   - On error: console.warn — never throw

6. queryLlmAllowlist(categoryId: string, normalizedInput: string): Promise<{ canonical_name: string; description: string } | null>
   - Guard: if (!supabase) return null
   - .from('llm_allowlist').select('canonical_name, description').eq('category_id', categoryId).eq('input_normalized', normalizedInput).maybeSingle()
   - On error: console.warn and return null
   - Wrap in Promise.race with 3-second timeout that resolves to null on expiry

All console.warn calls use the '[Supabase]' prefix to make them easily greppable.
